Method Endpoint Status Description Priority
Auth
POST /api/v1/auth/login Done Đăng nhập với email/password, trả về access + refresh token
POST /api/v1/auth/refresh Done Refresh access token bằng refresh token
Users
POST /api/v1/users/signup Done Tạo tài khoản mới
GET /api/v1/users/me Done Lấy thông tin user hiện tại
PUT /api/v1/users/me Done Cập nhật profile (full_name, phone)
POST /api/v1/users/me/qr-code Done Upload ảnh QR code thanh toán
GET /api/v1/users/search?email=X Done Tìm user theo email để mời vào event
Events
GET /api/v1/events Done Danh sách event của user hiện tại
POST /api/v1/events Done Tạo event mới
GET /api/v1/events/{id} Done Lấy chi tiết event
PUT /api/v1/events/{id} Done Cập nhật event (name, currency)
DELETE /api/v1/events/{id} Done Xóa event (chỉ creator)
Members
GET /api/v1/events/{id}/members Done Danh sách thành viên trong event
POST /api/v1/events/{id}/members Done Thêm member bằng user_id
POST /api/v1/events/{id}/members/by-email Done Thêm member bằng email
DELETE /api/v1/events/{id}/members/{user_id} Done Xóa member khỏi event
Expenses
GET /api/v1/events/{id}/expenses Done Danh sách chi tiêu trong event
POST /api/v1/events/{id}/expenses Done Tạo expense mới (payer_id + splits)
DELETE /api/v1/events/{id}/expenses/{expense_id} Done Xóa expense (chỉ người tạo)
Balances
GET /api/v1/events/{id}/balances Done Lấy balance của tất cả thành viên
GET /api/v1/events/{id}/balances/simplify Done Tính toán lại debts để số lần thanh toán ít nhất (minimum cash flow) HIGH
Invite
POST /api/v1/events/{id}/invite Done Tạo invite code/link để chia sẻ event MEDIUM
POST /api/v1/events/join/{code} Done Tham gia event bằng invite code MEDIUM
Settlements
GET /api/v1/events/{id}/settlements Done Danh sách settlements trong event
POST /api/v1/events/{id}/settlements Done Ghi nhận một lần thanh toán (bao gồm qr_code_url)
Stats
GET /api/v1/events/{id}/stats Done Thống kê event: tổng chi, số expense, số member LOW
Utils
GET /api/v1/utils/health-check/ Done Health check cho Docker
